Savage Worlds Web Tools
Flexible Dice Roller

Roll some Savage Dice. This little tool should be able to handle pretty much any combo you throw at it. It'll even handle damage rolling and wound calculation for you if you wish.

Dice Designate your Trait roll with an * for wild die rolling, just about anything (logical) can be put here
Examples
  • d8 add 2: d8* + 2 (trait test)
  • d8 with wild die + d6: d8 + d6 (damage test)
  • d12 with wild die subtract 4: d12* - 4 (trait test)
  • d12 but substract d6 then add 6: d12 - d6 + 6 (damage test)
  • You can now use multiplication (x) and division (/) as well as adding and subtracting. NOTE: This system doesn't use Algebraic Order of Operations (yet, this is planned as are parenthesis and exponents)
